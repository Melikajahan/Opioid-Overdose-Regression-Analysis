# ZIP-level data (restricted)

ZIP-level analytic data used in the manuscript are not included in this repository.

The restricted ZIP-level dataset includes opioid-related emergency department visit outcomes and non-public demographic, socioeconomic, community, healthcare-access, treatment-access, and naloxone-access variables. These data are not suitable for open posting under the project’s data-use and confidentiality constraints.

This folder is intentionally limited to documentation. Authorized users with access to the original ZIP-level data may use the manuscript, appendices, and repository documentation to understand the ZIP-level analysis workflow, subject to applicable data-use agreements.

No restricted ZIP-level data values should be committed to this repository.
