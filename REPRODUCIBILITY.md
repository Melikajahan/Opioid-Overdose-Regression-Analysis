# Reproducibility Notes

The county-level analysis can be reproduced using the county-level data included in this repository.

The ZIP-level analysis cannot be fully reproduced from public files because the ZIP-level analytic dataset is restricted or non-public. This includes the opioid-related emergency department visit outcome data and several demographic, socioeconomic, community, healthcare-access, treatment-access, and naloxone-access variables.

Where possible, the repository provides documentation and selected non-disclosive outputs to support transparency. Authorized users with access to the restricted ZIP-level data may reproduce the analysis locally, subject to applicable data-use agreements.
